# ZenSidebar - 极简侧边栏标签与书签管理器

`ZenSidebar` 是一款为全屏浏览和极致体验设计的 Chrome 扩展程序。它通过一个灵动的悬浮球召唤侧边栏，集成了标签切换、书签管理以及智能地址栏搜索功能。

## ✨ 主要特性

- **💎 毛玻璃美学 (Glassmorphism)**: 采用现代感十足的半透明磨砂背景，并支持系统级亮色/暗色模式自动适配。
- **🚀 智能地址栏**: 仿 Chrome 体验，支持网址直达、关键词搜索，并能实时联想您的历史记录与收藏夹。
- **📂 深度书签集成**: 专注展示“书签栏”内容，支持多级文件夹折叠交互，并配备原生网站图标 (Favicon) 展示。
- **🧭 灵动悬浮球**: 
  - **隐身模式**: 平时仅显示图标，悬停时展现完整交互界面。
  - **自由拖拽**: 随心调整位置，并具备自动视口吸附功能，绝不丢失。
- **🛡️ 纯净隔离**: 基于 Shadow DOM 技术实现，保证插件样式与网页内容互不干扰，安全稳定。
- **🏷️ 标签管理**: 垂直排布已打开的标签页，支持快速切换、一键关闭，并智能过滤系统限制页面。

## 🛠️ 安装指南

1. **下载项目**: 克隆或下载本代码库到本地。
2. **打开扩展程序**: 在 Chrome 浏览器地址栏输入 `chrome://extensions/`。
3. **开启开发者模式**: 点击页面右上角的“开发者模式”开关。
4. **加载插件**: 点击“加载已解压的扩展程序”，选择本项目所属的目录。
5. **开始体验**: 在网页左侧找到悬浮图标或使用快捷操作。

## ⌨️ 快捷操作

- **单击悬浮球**: 展开/收起侧边栏。
- **拖拽悬浮球**: 调整球体位置。
- **地址栏输入**:
  - `Enter`: 访问网址或搜索。
  - `↑/↓`: 在搜索联想选项中快速切换。
  - `Esc`: 关闭联想列表。

## 📦 项目结构

- `content.js`: 核心逻辑，负责 Shadow DOM 注入、UI 渲染与交互。
- `background.js`: 后台服务，处理标签查询、书签搜索及历史记录联想。
- `manifest.json`: 插件配置与权限管理。
- `smartpark.png`: 悬浮球图标素材。
